This is a neural network built with PyTorch that generates images based on the MIT Pizza dataset. The architecture used is a Generative-Discriminator Network. 

Project for high school 11th grade computer science class.

It was originally created in 2022, uploaded to GitHub in 2024. The notebook metadata was updated in 2026 so it can be viewed correctly on GitHub, therefore to view it online choose the updated version.

The original file can still be used when downloaded and opened in any code editor.
